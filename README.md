# UCLA Bruin Custom Posts
Brings custom post types to enhance WordPress for the UCLA community. 

## Custom Post Types
1. people
1. projects
1. publications
1. events
1. resources

Best used in conjunction with Advanced Custom Fields to develop structured content schemas.
